# Modes

## Throne Mode

The player rules a polity directly. Decrees can affect policy, economy, armies, settlement founding, infrastructure, trade, and diplomacy. The plausibility gate checks each decree against world logic before it enters the chronicle.

## Providence Mode

The player cannot order settlements directly. Influence is supernatural and indirect through blessings, curses, disasters, omens, and free-form divine acts. Every act records an Omen Log trail.

## Overview

Overview shows polities and settlements together, with mode switching into Throne or Providence when the scenario allows both roles.
