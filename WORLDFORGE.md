# World Forge

World Forge is pre-game authoring, not live divine intervention.

Creators can:

- Paint terrain and biomes.
- Draw region borders.
- Place polities and settlements.
- Add region tags for organization.
- Add polity and settlement tags that become prompt context.
- Write a concise world briefing.
- Set world logic rules for magic and technology.
- Publish, copy, and version worlds through the Community Hub.

Briefings should be terse and commanding. World logic rules should be concrete because they drive decree plausibility and Providence grounding.
